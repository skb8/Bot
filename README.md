# Telegram Bot for Tracking Titles

Бот для Telegram на Python, предназначенный для отслеживания тайтлов по их уникальным ID.

## Возможности
- **Ограничение доступа**: Бот отвечает только пользователям, чьи Telegram User ID указаны в файле конфигурации.
- **Интерактивный интерфейс**: Управление списком отслеживания происходит с помощью Inline-кнопок прямо в чате с ботом.
- **Отказоустойчивость**: Бот автоматически восстанавливает соединение при обрывах сети или ошибках API Telegram (`ConnectionResetError`, `ProtocolError`).
- **Локальное хранение**: Данные пользователей сохраняются в файл `trackers.json` (который автоматически добавлен в `.gitignore`).

---

## Требования
- Python 3.11+
- Виртуальное окружение (рекомендуется)

---

## Установка и запуск

1. **Клонируйте репозиторий**:
   ```bash
   git clone https://github.com/skb8/Bot.git
   cd Bot
   ```

2. **Создайте и активируйте виртуальное окружение**:
   ```bash
   python3 -m venv venv
   source venv/bin/activate  # Для Linux/macOS
   # или
   venv\Scripts\activate     # Для Windows
   ```

3. **Установите зависимости**:
   ```bash
   pip install pyTelegramBotAPI python-dotenv
   ```

4. **Настройте переменные окружения**:
   Создайте файл `.env` на основе примера `.env.example`:
   ```bash
   cp .env.example .env
   ```
   Откройте `.env` и укажите ваш токен бота и список ID пользователей через запятую:
   ```env
   BOT_TOKEN=123456789:ABCdefGhIJKlmNoPQRsTUVwxyZ
   ALLOWED_USERS=123456789,987654321
   ```

5. **Запустите бота**:
   ```bash
   python bot.py
   ```

---

## Использование
1. Отправьте команду `/start` боту в Telegram.
2. Бот отобразит список ваших отслеживаемых ID (если они есть) с кнопками «❌ Удалить ID: <id>» для удаления.
3. Нажмите «➕ Добавить ID тайтла» и отправьте нужный ID в следующем сообщении, чтобы добавить его в список.
